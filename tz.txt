KeyboardNinja.

Программа под Linux. 

Планы реализации программы:

Минимум:
После запуска программы пользователь должен написать "run"(и нажать "enter") для начала игры. После введения этой команды, на экран будет выведено одно слово(eng). Пользователь должен набрать его быстро и без ошибок и нажать "enter". Далее выводится другое случайное слово(eng) и действия пользователя повторяются, пока пользователь не напишет "end"(нажав после этого "enter"). После введения "end"( и после "enter") на экране терминала высвечиваются результаты:
correct words: <число правильно введённых слов>(int)
uncorrect words: <число неправильно введённых слов>(int)
time: <время в секундах>(double)

Откуда берутся слова?
Из текстового файла input.txt. Этот файл задан авторами и не может быть задан пользователем.

Какая структура текстового файла?
В текстовом файле слова(строки) стоят столбиком //и в каждой строке, для одинаковой длины, добавлены пробелы(или если слово имеет максимальную длину, из всех остальных слов, то пробелов не имеет). Это сделано для качественной работы функции fseek()//. После слова стоит "\n". Примерно структуру можно представить так:

...
abandon<\n>
abate<\n>
abduct<\n>
...

Слов в текстовом файле чуть больше 4500. Слова разные. Повторов нет.

Как программа выбирает слово? 
С помощью функции getrand(). Она(функция) генерирует случайное число и это случайное число присваивается переменной r. Далее программа через цикл перебирает слова по порядку, пока не дойдёт до слова, которое по счёту в столбе сходится с переменной r. Допустим функция рандома сгенерировала цифру 3. Работу выбора слова можно представить так:

abandon  <---- Допустим это нулевое(по счёту) слово в файле input.txt (Заранее установленный счётчик == 0) Программа проверит равен ли счётчик цифре рандома. В нашем случае r == 3, счётчик == 0, поэтому программа прибавит к счётчику единицу.
abate <---- Это первое(по счёту) слово. 1 != 3. счётчик++
abduct <---- Второе слово. 2 != 3. счётчик++
hello <---- Третье(нужное слово) т.к. 3 == 3. Это слово выведется на экран.
...

Как засекается время? 
После команды "run" таймер стартует.
После команды "end" таймер останавливается.

Как пользователю ввести слово?
Написать на клавиатуре заданное слово(все буквы маленькие, при смене регистра будет засчитано неправильное слово) и нажать "enter". Слова английские, раскладка клавиатуры должна быть "ENG".
